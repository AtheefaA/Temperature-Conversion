# Temperature Conversion

This project is a simple program that converts temperature
values between Celsius, Fahrenheit, and Kelvin.

## Features
- Converts multiple temperature units
- Simple and easy-to-use logic
- Beginner-friendly project

## Technologies Used
- Python / Java (based on your implementation)
- Console-based application

## How It Works
The program takes temperature input from the user,
applies conversion formulas, and displays the result.

## Example
Input: 30°C  
Output: 86°F
